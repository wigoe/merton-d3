Merton Structural Credit Model (Interactive)

This page visualizes the classic Merton (1974) structural model of credit risk.
Equity is treated as a call option on firm assets with strike equal to the face value of debt.

Panels show:

1. Equity vs Asset Value
2. Equity Volatility vs Equity
3. Default Probability vs Equity
4. Credit Spread (or Yield) vs Equity

The interface allows users to explore leverage, volatility shocks,
and calibrate asset value to observed market equity.

Built with D3.js — no backend required.
